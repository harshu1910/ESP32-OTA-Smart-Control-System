
# Demo Videos

## Before OTA Update

This video demonstrates the initial firmware behavior of the ESP32 OTA Smart Control System before performing a wireless firmware update. The hardware components including LED, buzzer, and servo motor operate with the original firmware configuration.

## After OTA Update

This video demonstrates successful Over-The-Air (OTA) firmware deployment on the ESP32 without using USB flashing. After the wireless firmware update, the modified firmware behavior can be observed through faster LED blinking and updated hardware response.

## Demonstrated Features

* Wireless OTA firmware updates
* ESP32 web server interface
* Real-time embedded hardware control
* Servo motor operation
* LED status indication
* Buzzer notification system
* Remote firmware flashing workflow
